 You know how in NewerSMBW Worlds after 9 are then the alphabet? (World A, World B, World C, etc)

I think you can actually remove that function, or change what World starts counting like that.
In the "newer.cpp" file, there is a line like so:
dest[0] = (world >= 10) ? (world-10+'A') : (world+'0');
First of all, 'dest[0]' is the World number, 'dest[1]' is the dash, 'dest[2]' is the first letter of the course ID, and 'dest[3]' is the second. I believe 'dest[4]' is an unused value, since it's set to equal 0 which is basically nothing.

I'm no expert, but I think if you replace both of the '10s' in (world >= 10) ? (world-10+'A') with a different number, like 18, then the letters will only show when the World # from Levelinfo.bin is 18 or higher. I'm planning on testing this out in the future.

That edited line would look like this:
dest[0] = (world >= 18) ? (world-18+'A') : (world+'0');
If you wanted to remove the letters from the World ID, you would just need to replace that line with this:
dest[0] = (world+'0');

This is all theory, and isn't tested yet.

 I've found where and how you can modify the items that appear in the shops (in NewerSMBW).

In the shop.cpp file, if you scroll a little down, you can see what files the game loads
from the "lakitu.arc" file. It's as follows:
"g3d/yoshi.brres", "g3d/desert.brres", "g3d/mountain.brres", "g3d/sakura.brres", "g3d/santa.brres", "g3d/ghost.brres", "g3d/space.brres", "g3d/koopa.brres", "g3d/sewer.brres", "g3d/goldwood.brres"
Files "sewer.brres" and "koopa.brres" aren't actually in the lakitu.arc file. This means this is some unused
code. Also, space.brres it the exact same as "yoshi.brres". But anyways, you could add more models in the world
order you want them to be used.

If you scroll past the halfway point, there will be a section showing the IDs of what show up in the shop.

For example, in Yoshi's Island, it's shop has:
-A Mushroom for 1 Star Coin
-A Fire Flower for 2 Star Coins
-An Ice Flower for 2 Star Coins
-A Propeller for 3 Star Coins
-2 Fire Flowers & 1 Ice Flower for 5 Star Coins
-2 Mushrooms, A 1-Up, & 2 Propellers for 8 Star Coins.
Those items like this in code:
MUSHROOM, FIRE_FLOWER, ICE_FLOWER, PROPELLER, FIRE_FLOWER, ICE_FLOWER, FIRE_FLOWER, MUSHROOM, MUSHROOM, ONE_UP, PROPELLER, PROPELLER
You can change what item is where by changing the name. This is what each item is in this code. I also put their IDs for... something I'm not sure of.
-MUSHROOM = Mushroom = 0
-FIRE_FLOWER = Fire Flower = 1
-ICE_FLOWER = Ice Flower = 2
-PENGUIN = Penguin Suit = 3
-PROPELLER = Propeller Mushroom = 4
-MINI_SHROOM = Mini Mushroom = 5
-STARMAN = Super Star = 6
-HAMMER = Hammer Suit = 7
-ONE_UP = 1-Up Mushroom = 8

This is how it functions. I'll start with the first line.

The first item (commonly Mushrooms) is 1 Star Coin.
The next 2 on the list (often Fire & Ice Flowers) are 2 Star Coins.
The 4th item (Usually either Propeller, Penguin, or Hammer Suit) is 3 Star Coins.

The 2nd line has the 3 items you can get for 5 Star Coins.

The 3rd and final line has the 5 items you can get for 8 Star Coins.

 What CLF (Customlevelfan) is true.

I don't know of any custom symbols after Z. I'd need to look into that.
I just wanted to mention that the # symbol in Newer's Message blocks is a filled Blue Block
(Like NSMBW's Red Blocks, but it's Blue).
I'll research more of these symbols later. I think I know all of them, actually.

So I found something interesting. Newer's Messagefont file is like an amalgamation (combination)
of NewSMBW's Message, and number font. Maybe a little bit of mariofont, I don't know. Anyways, the
"Blue Block" image replaces the location where, in theory, the # symbol would be. Looking to the
right of Blue Block image are some other images in place of some text symbols. Here's a little list
of what image most likely replaces what symbol in Newer's Message Blocks.

# = Blue Block
$ = Yellow Block
% = Green Block
& = Question Block
( = Toad House
) = Another Toad House
* = Another Toad House... (Literally all the same)
+ = Purple Course Icon (Usually indicates if a course has 2 exits, and only one has been completed in Newer)
- = Shop Course Icon (Represents Shops in Newer)
