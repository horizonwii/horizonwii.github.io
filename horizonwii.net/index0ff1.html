This is literally better than John Telstra's (from RVLution) version of the tutorial, since it allows longer text.

Requirements

-NewerSMBW and NewerSMBW sources

-A text editor

-A USB or SD Device to launch NewerSMBW

-Riivolution and Wii or Wii U

Tutorial

Step 1: Download NewerSMBW at the Newer Team’s Website and wait for it to download and extract it to your SD card or USB device.

Step 2: Make sure you've set up the "compiling" of the NewerSMBW sources if you haven't already.

Step 3: Get a text editor, usually there's one built into Mac OS X and Linux. If you're on Windows, I would recommend you use PSPad, download it here.

Step 4: In the NewerSMBW sources folder, open the file Kamek/src/randomcrap.S with your text editor.

Step 5: Use the "Find" feature in your text editor (in PSPad on Windows, you simply hit "Ctrl-F" then have the “As Text” box checked). Type Whoops and you should see this line:
.string "Whoops! Newer 1.0.2 has crashed.\nPlease take a screenshot of the info below\nand send it to newer@newerteam.com.\nYou can scroll through this report using the D-Pad.\n"
And edit the string to something you can think of (but do not edit the UniLogStr:, #WillAlloc, #Alloced, #WillFree, and any other lines), for example:
.string "******** A CRASH OCCURRED! ********\n/- Please give us more information about the\ncrash (including a \"screenshot\" of it) and\nsend it to newer@newerteam.com.\nUse the D-Pad to scroll through the\ninformation/report. To get out of the crash,\nplease turn off your Wii/WiiU console by holding\ndown the POWER button or unplug your Wii/WiiU. -\\\n"
There has to be quotes surrounding the string, do not remove the quotes.
And there has to be an newline (\n) at the end of the string, so the exception info will display properly.

If you want to have a newline character inside the string, use \n instead.
If you want to have a left slash character inside the string, use \\ instead.
If you want to have a double quote character inside the string, use \" instead.


Step 6: Now, you will need to find a way trigger an exception in NewerSMBW. To speed the process up here, John Telstra (on RVLution) made something that can trigger it. Download it here: https://www.dropbox.com/s/z36gm0j9j2wlp53/ExceptnTriggerer%28Newer%29.zip?dl=0
NOTE: Remember to look at the readme.txt file in the ZIP archive before launching NewerSMBW. And backup your /save folder on your SD card or USB device, before downloading the ZIP file.
