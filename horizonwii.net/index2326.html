 The game's files are essential to work with Reggie, and you can't use it without them. The extracted files may differ between dumping methods, but what you'll need will be there.

Method 1 : ReggieDumper

I recommend this method because it's the easiest and most legal one.

Requirements :

- Homebrew Channel
- Original NSMBW Disc (any region does the job)
- SD Card (possibly not SDHC) or USB drive with at least 400MB of free space (to dump the whole disc), or 15 MB (for the Stage folder only).
- ReggieDumper


Steps :

Download ReggieDumper and extract the archive content on the root of the SD Card/USB Drive. Overwrite if necessary.



Turn the Wii on and insert the NSMBW disc BEFORE starting ReggieDumper. After that, open up the Homebrew Channel and start the application. What you will see is this.



Now press A if you want to only dump the files needed for Reggie, or press B to get a full disc dump, and wait until it's done. When it finishes, press Home to go back to the HBC.



Now put the SD Card/USB Drive back into your PC. On it will find a folder named "reggie", with the dumped files inside. You can move it wherever you want.



Method 2 : WiiXplorer

WiiXplorer is a file explorer for the Wii. It allows browsing Wii Discs, which is what is needed to dump the game.

Requirements :

- Homebrew Channel
- Original NSMBW Disc (any region does the job)
- SD Card or USB drive with at least 400MB of free space
- WiiXplorer

Steps :

Download WiiXplorer and extract it on the SD Card/USB Drive.



Turn the Wii on and insert the NSMBW disc BEFORE starting WiiXplorer. After that, open up the Homebrew Channel and start the application. What you will see is this.



Now press the SD Card symbol on the top left and choose Disc. Pay attention not to switch back and forth between the two too quickly, or the app will crash in an awful way.



After it finishes loading, click on any folder to make the disc's folders appear. You will see two of them, named respectively named "0 Update" and "1 Data". Press + on the latter and choose Copy, then switch back to the SD Card/USB Drive, press + anywhere and choose Paste.



After copying, press Home to exit the app/turn off the Wii.

Method 3 : Wii Backup Fusion

Also known as WiiBaFu, it works on Windows, Mac and Linux. Being based on WIT, it supports several formats and has many useful features which other tools don't have. It's also much faster than WiiScrubber, but it doesn't support extracting single files.

Requirements :

- NSMBW ISO (can be scrubbed or not), WBFS or CISO. You can get one using CleanRip.
- Wii Backup Fusion

Steps :

Download and extract the archive wherever you want and start the program, then click the Load button on bottom-left.



Browse the folder where the ISO file is (NOT the ISO file itself), then wait for the program to finish scanning the chosen folder. At the end the games in it will appear in the list. Click on NSMBW and then on Extract Image.



A new window will appear. Click on Open and choose a folder for the files to be extracted in, then press OK.



Wait for the extraction to complete, then you can close WiiBaFu.

Method 4 : WiiScrubber (OUTDATED)

WiiScrubber only works on Windows, you must use either Method 1 or 3 if you're on OS X or Linux. This method also allows to extract single files if needed (not folders though).

Requirements :

- NSMBW 1:1 ISO file (which means the 4.37 GB one). An excellent (and legal) way to get one is CleanRip.
- WiiScrubber


Steps :

Download the WiiScrubber pack from the link above and extract it. The only needed programs are MakeKeyBin and WiiScrubber itself, the others can be deleted. Open MakeKeyBin.exe, type 42 and click Generate, press OK and close the program.



Start WiiScrubber, click on Load ISO and choose the NSMBW ISO.



After it loads, right-click on the partition named DATA (which is usually 1 if an update partition is present), then choose Unencrypted Partition > Extract.



Choose the destination folder and wait for the extraction to finish (it may take some time). When it ends, you can close WiiScrubber (if a message appears, ignore it and click on Yes).

Method 5 : Download

This method will only provide you the folders required by Reggie/Reggie Next. It's a really lazy one, so make sure to only use it as a last-time resort.

Simply download this, unpack it wherever you want and you're set

Optional Step : Remove Write Permission to Dump Folder

Putting the dump folder in read-only mode can be useful to avoid accidentally editing the files inside it, therefore keeping the dump intact.

To do this, right-click the dump folder and choose Properties. After that, enable the Read-only option in the window that appears. Make sure it's checked, and not just a round dot (which means only part of the folder is read-only).



After that, click Apply and then OK to save the change.
