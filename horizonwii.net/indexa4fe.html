Agreed, and another issue is, that you can't replace any sound effect (you'll not be able to save you're changes), if:

If the sound effect (you're using for replacing) length (in samples) is longer or smaller than the original.
If the sample rate/frequency (in Hz) of the sound effect is not equal to the one of the old sound effect.
If the channels (Mono[1]/Stereo[2]) of the sound effect is not equal to the one of the old sound effect.

Posted by Mayro
It's pretty simple really - put the contents of a zip on an SD card.
Make sure to put it on the root of the SD card or USB device.

Just keep in mind that there should be riivolution folder (with an XML file) and the hack's patch folder (where all the hack's files/etc. are at) in the ZIP file or whatever of the hack, if you want to play it with Riivolution.

If there isn't an riivolution folder (with an XML file) in the ZIP file or whatever, of the hack and just the data files or such, then it's possible that the hack isn't designed for use with Riivolution.
