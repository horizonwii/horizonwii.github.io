According from what I've seen and know, the World Name of a map in the LevelInfo.bin corresponds to the World Name only in the Star Coin Menu, it doesn't correspond to the World Name of the actual map.

The World Name of the map in the LevelInfo.bin will show only in the Star Coin menu, while the actual name (you changed in Map --> World Editor... in Koopatlas) of the map will show only in the file select screen, and on the "HUD" (heads-up dock) when you're inside the map.
