Posted by John Telstra
What up, ladies and gents?


Have you ever wanted to change the save name in NewerSMBW (you know, the one in /save/SMNE/NerSMBW.sav, or /save/SMNJ/NerSMBW.sav, or /save/SMNP/NerSMBW.sav)? Well your wish has just came true!

I’ve heard someone in some forum posted for help on how to change the save name in NewerSMBW.
What he said:

Hi. Need help. In the .sav file of NewerSMBW calls NerSMBW.sav but the NSMBW original is wiimj2d.sav. How to change to NJDMBW.sav (only NJDMBW)?

Requirements

- A text string to hexadecimal converter.
- NewerSMBW's XML w/ the line: <savegame external="/save/{$__gameid}{$__region}" clone="false"/>
- An XML editor (MS Notepad. PSPad, Notepad++, etc)
- NewerSMBW
- An SD card at least 1 or 2 GB of space or a USB Stick.
- A Wii or Wii U console.

Tutorial

Step I: Download NewerSMBW at the Newer Team’s Website.


Step II: Get a text string to hexadecimal converter, I would recommend String Convertor v1.4. since it’s free and simple. Download it here. And extract the ZIP File into one folder. Now your Anti-Virus protection may not want you to download Windows applications (or .EXE files) from sites/etc. But don’t worry, this app is not a virus.


Step III: Once you’re done extracting the ZIP File, go to the folder where you extracted the program, And run the .exe file. Have these boxes in the program checked:



Usually it’ll automatically be checked, but if it doesn’t, just check them.


Step IV: Put the text you want in the ASCII String box in the program, It has to be at least 7 characters.
DO NOT PUT IN WORD THAT’S LONGER THAN 7 CHARACTERS! Otherwise the game will not accept the name and it’ll use the old save name, wiimj2d !

Also do not add the .sav extension at the end of the name! And you cannot remove the .sav extension into some other file extension. The .sav extension is mandatory - you can't remove it.

Also, you cannot put shorter text for the save name. Because putting the word “test” (which is at least 4 characters) (w/o quotes) the save name will not be test.sav when you launch the game, it’ll instead be testj2d.sav or some other random name when you see it, as it overwrites every character (in the original save file name) it uses!


After you put the word up, click the “To Hex” button in the program:



Then, the hex offset should appear in the hex box in the program. So the hex offset of the word “Example” would be hex offset, 4578616D706C65.

Now don’t close the program, leave it open. Now, get an XML editor, I would recommend PSPad since it’s free and simple, download PSPad (at the bottom of the thread). And extract it into one folder.

Step V: BACKUP YOUR NEWERSMBW XML BEFORE EDITING IT! Open the NewerSMBW XML w/ PSPad, find these lines in the XML:


<!-- Save name change -->
<!-- PAL--><memory offset="0x802F148C" value="4E6572534d42" original="7769696D6A3264"/>
<!--NTSC--><memory offset="0x802F118C" value="4E6572534d42" original="7769696D6A3264"/>
<!--JPN--><memory offset="0x802F0FAC" value="4E6572534d42" original="7769696D6A3264"/>




Copy the hex value in String Convertor v1.4. and paste in the highlighted text as it shows in the picture:



NOTE: DO NOT EDIT ANY OTHER SETTINGS IN THE MEMORY OFFSET LINES IN THE XML, OR YOU YOU MIGHT GET THIS MESSAGE HERE:



OR SOME OTHER ERROR!

After pasting the value, it should look like this:



Step VI: Hit Ctrl-S on your keyboard to save the XML.
Delete the save folder on the SD or USB Device.
BUT BE SURE TO backup the save folder on your SD or USB device before deleting it from your SD or USB device!


Step VII: Insert your SD or USB device in your Wii. Launch NewerSMBW up. And when it prompts you that the save file has been create successfully, click OK on the window:



When the screen goes black, turn off your Wii, then remove the USB or SD device from your Wii.

Step VIII: Plug in the SD or USB device in your computer. Open the root of the device on your computer, browse to /save/SMNE/ (or SMNP if you have PAL version of the game, or SMNJ if you have the Japanese version of the game), and then your save name mod change should appear here:




This tutorial/etc. is 100% Tested.
