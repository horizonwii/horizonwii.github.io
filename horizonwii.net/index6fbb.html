 These are the level IDs used in NSMBW. You must use these if you want your level to replace the correct stage in-game.

1 to 8 - Normal Level (9 is possible, but unused)
20 - Coin Battle Level
21 - Ghost House
22 - Tower
23 - Tower 2 (unused, icon is the same as normal tower)
24 - Castle
25 - Castle 2 (unused, icon is the same as normal castle)
26-27-28-29 - Toad House
33-34-35 - Ambush
36 - Cannon
38 - Ship
40 - Title
41 - Peach's Castle
42 - Credits

NewerSMBW has different rules, so i added these too on request. Please note that in the extra worlds and unused levels, these are not followed at all and you may get confused. In such case, use the levelnames.xml file in Reggie Next's Newer patch.

1 to 10 - Normal Levels
11-12-13-14 - Titlescreen (except Corrupted Game)
15 - Tower
16 - Tower 2
24 - Final Castle
25 - Castle
31 - Switch Palace
33 - Challenge House
34 - Music House
38 - Airship
40 - Corrupted Game Titlescreen
41 - Yoshi's House, Cutscenes, Credits
99 - Shop

Note : If you're making your own Newer hack, using these rules for your level filenames is optional, as you can easily use any level number you want (provided it's between 01 and 99) by editing the level entry in LevelInfo.bin.

Note 2 : Koopatlas and LevelInfo.bin use different numbers for displaying level names. You can find those here.

 I got some extra information from the newer.cpp file for NewerSMBW.
These IDs are for the Display Names in LevelInfo.bin file.

21, Tower
22, Tower 2
23, Castle
24, Fortress (Generally used for Special Worlds)
25, Final Castle
26, Train
27, Airship
28, Switch Palace (Unused?)
29, Yoshi's House
30, Key 1 (Unused?)
31, Key 2 (Unused?)
32, Key 3 (Unused?)
33, Key 4 (Unused?)
34, Key 5 (Unused?)
35, Key 6 (Unused?)
36, Music House
37, Shop
38, Challenge House
39, Red Switch Palace
40, Blue Switch Palace
41, Yellow Switch Palace
42, Green Switch Palace

This is the source of my information, by the way.
// 0-19 are handled by code
// To insert a picturefont character:
// \x0B\x01YY\xZZZZ
// YY is the character code, ZZZZ is ignored
L"A", // 20, alternate
L"\x0B\x0148\xBEEF", // 21, tower
L"\x0B\x0148\xBEEF" L"2", // 22, tower 2
L"\x0B\x012E\xBEEF", // 23, castle
L"\x0B\x012F\xBEEF", // 24, fortress
L"\x0B\x013D\xBEEF", // 25, final castle
L"\x0B\x014D\xBEEF", // 26, train
L"\x0B\x0132\xBEEF", // 27, airship
L"Palace", // 28, switch palace
L"\x0B\x0147\xBEEF", // 29, yoshi's house
L"\x0B\x014E\xBEEF" L"1", // 30, key 1
L"\x0B\x014E\xBEEF" L"2", // 31, key 2
L"\x0B\x014E\xBEEF" L"3", // 32, key 3
L"\x0B\x014E\xBEEF" L"4", // 33, key 4
L"\x0B\x014E\xBEEF" L"5", // 34, key 5
L"\x0B\x014E\xBEEF" L"6", // 35, key 6
L"\x0B\x0138\xBEEF", // 36, music house
L"\x0B\x0133\xBEEF", // 37, shop
L"\x0B\x0139\xBEEF", // 38, challenge house
L"\x0B\x0151\xBEEF", // 39, red switch palace
L"\x0B\x0152\xBEEF", // 40, blue switch palace
L"\x0B\x0153\xBEEF", // 41, yellow switch palace
L"\x0B\x0154\xBEEF", // 42, green switch palace
