 Since Danster didn't know and 9211tr's guide being confusing, i wrote this one myself.

Requirements :

- The tileset you want to edit
- BrawlBox

Note :

- With this method, you CAN'T change the slot. For that, you need to open the tileset in Puzzle and use the "Set Tileset Slot" option, and save again.

Steps :

- Rename the tileset to whatever name you have (it's advised to follow the Nintendo naming scheme, which is PaX_whateveryouwant (where X is the tileset slot and spaces must be replaced with "_")
- Open the file in BrawlBox
- Go in every subfolder inside the tileset, and rename every file which contains the old name to reflect the filename change (right click > Rename)
- Save the file and you're done !
