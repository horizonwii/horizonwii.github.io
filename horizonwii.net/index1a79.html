 Another tip : to set a connected pipe, enable the related setting on the entrance and edit the path ID. On the exit, set the same things and enable Connected Pipe Reverse.

The exit direction is a must for horizontal pipes and other directions. The value should be set on the entrance, and not the exit. If both ends are enterable, set it on both for the other entrance.

Depending on the sprite, you may (or may not) have to do ASM hacks on NSMBW (or NewerSMBW) which is extremely hard to do, since it requires learning of PowerPC Assembly language and also:

  • If you're using NSMBW; Reverse engineering the main.dol file of the NSMBW disc and/or putting memory offset lines in your XML (if you use Riivolution).

  • Or, if you're using NewerSMBW; Modification to the NewerSMBW source code, and compiling it (a tutorial on how to set up the NewerSMBW sources is right here). And you may also need knowledge of C++ as well, too.

 Remove the coin sprites at the beginning (the ones just above the player) and move the entrance back a few squares. Then, move all of the fire bros on the tick tock girder things up by one or two squares. You may or may not need to get rid of the dripping sound creator, but you might need to. Lastly, go into zone settings and change both the upper and lower bounds to 0 instead of -32. That should fix your level.
