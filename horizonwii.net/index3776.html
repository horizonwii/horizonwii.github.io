The .LH file is just a compression to the .arc file, mainly to reduce space in the NewerSMBW mod.
So it isn't required, you can simply use the normal 01-01.arc (or whatever else file) in NewerSMBW, uncompressed, without the .LH compression.

Just don't rename the normal/uncompressed file (e.g. "01-01.arc") to put the .LH file extension in it (e.g. "01-01.arc.LH"), or the game will think it's compressed and will try to decompress it (and it'll fail/crash, since the file is uncompressed).
If you didn't rename the 01-01.arc file to 01-01.arc.LH, then you should look at the Level Troubleshooting Guide , for help.

1. Download NTCompress (this is my Google Drive).
2. Decompress the ZIP.
3. Put your ARC level (??-??.arc) into the same folder where compress.bat and ntcompress.exe are
4. Double click on compress.bat
5. Some files will pop up. One or two new items will show up.

If one new item shows up, that will be ??-??_LH.bin
6. Rename ??-??_LH.bin to ??-??.arc.LH
7. Use WiiScrubber and do your job.

If two new items show up, those will be ??-??_LH.bin and ??-??_LH_LH.bin
6. Delete ??-??_LH_LH.bin
7. Rename ??-??_LH.bin to ??-??.arc.LH
8. Use WiiScrubber and do your job.

Oh, ignore the ARC files in the stage folder please.

You'll need to download the level info editor from RoadRunnerWMC.
https://github.com/RoadrunnerWMC/Level-Info-Editor
You'll also need python and some other things. Those are explained when you follow the link.
You can simply get a binary/EXE download of that program without compiling anything or such, just get it here.

And then, follow this tutorial here to know how to use the level info editor.

Also, according from what I've seen and know, the World Name of a map in the LevelInfo.bin file corresponds to the World Name only in the Star Coin Menu, it doesn't correspond to the World Name of the actual map.

The World Name of the map in the LevelInfo.bin file will show only in the Star Coin menu, while the actual name (that you changed in Map --> World Editor... in Koopatlas) of the map will show only in the file select screen, on the "HUD" (heads-up dock), and possibly elsewhere when you're inside the map.
