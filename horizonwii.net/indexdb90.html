 This tutorial will teach you how to scan your XML for errors in writing.

Requirements :

- An XML file you want to check
- Notepad++
- XMLTools Plugin
- Riivolution Schema

Note : The <memory> tag has no checks for the presence of all the required parameters, so use the information in this thread to check it yourself.

Steps :

- After installing the XMLTools plugin, open the XML you want to scan in Notepad++
- Click on Plugins > XML Tools > Validate Now
- A new window will open
- Browse to the Riivolution Schema XSD
- If everything is correct, a popup will appear telling you the XML is valid, otherwise a window will appear listing the errors you've made

That's it !
