 This tutorial was made by Er-Rakunda, but since i couldn't retrieve it i remade it from scratch.

What is this needed for ?

This tutorial can be useful to create speeded-up versions of BRSTMs, to avoid the boring task of making another one from scratch.

Requirements :

- The BRSTM file you want to edit and a way to listen to it
- An hex editor (i use HxD)
- A program or site that can convert between hex and decimal values (i use this)

Steps :

- First of all, make a copy of the original BRSTM, just in case
- Open the copy you just made in the hex editor (i will use HxD as a reference)
- Click the "Find" tab and then on the "Go To Offset" (or use the Go To function in your hex editor)
- Make sure hex is selected and type 64, then go to that offset
- Usually the value will be "AC 44", but not always. That is the sound's sample rate converted to hex.
- Go to the converter, and convert the value back to decimal (in this case it will be 44100)
- Add about 6000 to the converted value, then convert it back to hex
- Back in the hex editor, replace the old value with the new one and save the file
- Now you can test the new BRSTM by opening it in BrawlBox (or whatever you have). If needed, repeat the steps until you've gotten the right sound.

 These are the level IDs used in NSMBW. You must use these if you want your level to replace the correct stage in-game.

1 to 8 - Normal Level (9 is possible, but unused)
20 - Coin Battle Level
21 - Ghost House
22 - Tower
23 - Tower 2 (unused, icon is the same as normal tower)
24 - Castle
25 - Castle 2 (unused, icon is the same as normal castle)
26-27-28-29 - Toad House
33-34-35 - Ambush
36 - Cannon
38 - Ship
40 - Title
41 - Peach's Castle
42 - Credits

NewerSMBW has different rules, so i added these too on request. Please note that in the extra worlds and unused levels, these are not followed at all and you may get confused. In such case, use the levelnames.xml file in Reggie Next's Newer patch.

1 to 10 - Normal Levels
11-12-13-14 - Titlescreen (except Corrupted Game)
15 - Tower
16 - Tower 2
24 - Final Castle
25 - Castle
31 - Switch Palace
33 - Challenge House
34 - Music House
38 - Airship
40 - Corrupted Game Titlescreen
41 - Yoshi's House, Cutscenes, Credits
99 - Shop

Note : If you're making your own Newer hack, using these rules for your level filenames is optional, as you can easily use any level number you want (provided it's between 01 and 99) by editing the level entry in LevelInfo.bin.

Note 2 : Koopatlas and LevelInfo.bin use different numbers for displaying level names. You can find those here.

 With this tutorial you will be able to edit both NSMBW's and Newer's Fonts. I know there's already a method to do this, but this one is MUCH easier.

Requirements :

- A Windows PC
- The font files in BRFNT format
- BRFNT2TPL, look here if you have an older version of Windows
- BrawlBox
- Any image editor
- BRFNTify Next (OPTIONAL) - To preview your edited BRFNT file

Notes :

To not create a corrupted BRFNT, the program has the following limitations :

- All the TPL images must have the same width and height.
- They must use the same format.
- The TPL image count must not change.
- They must not use a palette based format.
- They must not have mipmaps. Use the latest version of BrawlBox to show the correct mipmaps (old ones may show an incorrect number).
- The TPL file must not be compressed in any way.

Steps :

1) Drag and drop the BRFNT file you want to convert in BRFNT2TPL
2) The program will generate some TPLs and a VBFTA file. While the TPLs are the images, the VBFTA file is used to store the original BRFNT informations that are needed to convert the TPLs back.
3) Open the TPL/s in BrawlBox and extract their texture (Right Click > Extract)
4) Edit the image as you wish in your image editor.
5) Now replace each TPL's texture in BrawlBox and save them.
6) Select all the TPLs and the VBFTA created by the program and drag them inside BRFNT2TPL.
7) The program will convert them in a BRFNT
8) (OPTIONAL) Open BRFNTify Next, click on File > Open and select your BRFNT file to preview it and poke with it if you want to.
