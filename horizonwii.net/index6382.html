C++ Source Code Download (from Treeki, on GitHub)

NOTE: All credit goes to Treeki for this program, I don't hold any credit for it. This program is a command-line application. On Windows, please open up the Command Prompt (e.g. by searching/opening for "cmd.exe" in the start menu) and running the program from there.

To run the program, in the command prompt, type
LHDecompressor <LHFileName> <DecompressedFileName>
and your .LH file should be decompressed.

To automate the process, here's a Windows batch file to do this:
@ECHO OFF SET /P OUTFILENAME=Please enter the filename for the decompressed LH file: LHDECOMPRESSOR "%~1" "%OUTFILENAME%"
Open MS Notepad, copy the text above into Notepad, save the file. When saving the file, name it, DRAG_LH_FILE_TO_DECOMPRESS.BAT. Make sure to save the file in the same folder where the LHDecompressor program is.

With the batch file, you can drag any .LH file into the batch file to decompress the .LH file, without having to open up the Windows Command Prompt.

Also,
Posted by customlevelfan78
The root node has a few parameters that can be applied to it :

- Version (Required) : Usually you will use version 1, there's not much else to say
- Root (Optional) : This path will be used as the root for all the paths used within this XML. It can be useful to avoid writing long paths over and over. Make sure to put "/" before the folder name.
- Shiftfiles (Optional) : Set to true or false. This is REQUIRED if you are injecting new files into the game, as it will allow Riivolution to move them in.
There's also a Log (e.g. log="true") parameter, for enabling debug data to be output to the RiiFS cmd.
