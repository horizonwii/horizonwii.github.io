Posted by newsuperhackboys
But now, about the message ids. I've edited some messages in NewerSMBW using the Message Editor. The IDs in the editor are different than the ones in Reggie!. I've heard something about Messages.py but I have no idea what that is. Can someone tell me the IDs for the messages or try to help me find them?
In Reggie!, when you reference an ID using the Message Box sprite, you're required to add 256 to the ID (so for example, if the message ID is "10", the ID when referencing it in Reggie!, would be 10+256). That's the reason why the IDs in Reggie! are different from those of the NewerSMBW Message Editor.

If you want to know what ID the Message Box sprite is using in Reggie!, copy the ID into a calculator and subtract it by 256, and that should give you the ID (referenced in the NewerSMBW Message Editor) that the Message Box sprite is using.

Posted by customlevelfan78
- BMG files can be converted to txt using Wiimm's SZS Tools
- You can use BREFF Editor to edit BREFF files
- Wii Layout Editor can only parse BRLANs, not edit them
- You can edit BRSEQ files with Wexos'Toolbox
- You can use Notepad++ to edit CSV files, hex editor is a bit overkill
- You can compress/decompress NSMBW's LZ files using Wexos'Toolbox. NSMBW also uses LZ11 (for the .rel files and for CreditsBG.arc in Newer)
- REL files can be decoded with IDA like DOL files
All those infos are added (along with a few more, that I discovered through constant Google searches, etc.).

Posted by customlevelfan78
- BrawlBox can't edit NSMBW's BRSAR file, for currently unknown reasons
Umm, what? If that's true, then how come people have successfully been able to replace the sound effects in the .brsar file of NSMBWii? Did the newest BrawlBox version (v0.78) broke something so that it can't be edited?


AFAIK, the only time BrawlBox will refuse to save your changes (when replacing a sound effect in the .brsar file) is:

If the sound effect (you're using for replacing) length (in samples) is longer or smaller than the original.
If the sample rate/frequency (in Hz) of the sound effect is not equal to the one of the old sound effect.
If the channels (Mono[1]/Stereo[2]) of the sound effect is not equal to the one of the old sound effect.
