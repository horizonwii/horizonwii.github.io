Better Wii BRSTM BRSAR Patcher

Current version: 1.0.1.


This program can be used to patch BRSAR files with BRSTM files greater than 16 MB (16777215 bytes), although the maximum limit is up to 4 GB (4294967295 bytes).

Before asking me for any help/questions/etc. please look at the README.TXT file in the download, first, before posting/doing anything.
If you can't find what you need help looking for (and you're sure you read the README.TXT file, first), then post it here.

Source Code Download: https://github.com/9211tr/better-brsar-patcher
Releases/Binaries Download: https://github.com/9211tr/better-brsar-patcher/releases/

NOTE: A batch script is also added in the download to automate the process of patching the BRSAR file. See the README.TXT file in the download for more details.
