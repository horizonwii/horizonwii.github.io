Koopatlas Doodads
Koopatlas uses images to add to the environment. All the images
used in Newer can be found in this rar file!
Just one thing I want to point out; in case if a Doodad in a Koopatlas map isn't included in the RAR file, you can open the specified map in Koopatlas (that has the Doodad not included in the RAR file), and click File --> Export Doodads...

Also, you should add the BRFNT2TPL program to the list, as it converts a BRFNT to TPL and vice versa, allowing you to edit a BRFNT font in NewerSMBW/NSMBWii.
