Just note that if you're going to compile the NewerSMBW sources on VirtualBox or VMWare, you'll need to create an Ubuntu (or Debian) Linux virtual machine. And you'll need to get the Ubuntu (or Debian) Linux installer ISO image, and you'll have to install it on the virtual machine (Google for a tutorial on how to install Ubuntu or Debian Linux on VirtualBox or VMWare), and when the installation is finished, follow my tutorial on how to compile the NewerSMBW sources on Ubuntu/Debian Linux.

Also, keep in mind that VMWare is not free (so you'll need to pay cash in order to use it). VirtualBox, is however, free (and you don't have to pay cash).
