 I remembered I had this mod. It was lost when Rvlution was shut down.
So I thought I'd post it here! This was created by John10v10!

This mod lets you replace Mario, Luigi, Blue and Yellow Toad's model
(Normal, Fire, Ice, Penguin, and Propeller Suit) with a box of whatever
images you choose! It will get replaced with any 512 by 512 image you want
to replace it with. The download has 3 files, which are in the "Avatar" folder.
Avatar1, Avatar2, and Avatar3n4.

-Avatar1 is Mario
-Avatar2 is Luigi
-Avatar3n4 is Blue, and Yellow Toad (They share their model, and file)
-The NPC Toad is also included in Avatar3n4. He only uses one texture.

The download also includes an useless Read Me XML used for loading the
3 Avatar files from the Avatar folder. Don't change the name of the 3 files, unless
you don't want to use the Avatar XML. It's your choice!

This can get pretty... "interesting" if you replace them with random images
from the internet. I don't have any previews because all of the footage is
gone.

I did make two mods from this. One of them is a pack. The characters inside
are named as followed. Bootleg Mario, Bootleg Luigi, Bootleg Toad, and Bootleg
Toadette! Together, they are... The Bootleg Squad!
Oh, I also changed the other Toad... yeah.

I also did Bootleg Wario, Waluigi, Spike, and Spike. I don't really know why...

These mods are included in the download. I've added into the XML and the Avatar folder. Anyways, here it is!

 What CLF (Customlevelfan) is true.

I don't know of any custom symbols after Z. I'd need to look into that.
I just wanted to mention that the # symbol in Newer's Message blocks is a filled Blue Block
(Like NSMBW's Red Blocks, but it's Blue).
I'll research more of these symbols later. I think I know all of them, actually.

So I found something interesting. Newer's Messagefont file is like an amalgamation (combination)
of NewSMBW's Message, and number font. Maybe a little bit of mariofont, I don't know. Anyways, the
"Blue Block" image replaces the location where, in theory, the # symbol would be. Looking to the
right of Blue Block image are some other images in place of some text symbols. Here's a little list
of what image most likely replaces what symbol in Newer's Message Blocks.

# = Blue Block
$ = Yellow Block
% = Green Block
& = Question Block
( = Toad House
) = Another Toad House
* = Another Toad House... (Literally all the same)
+ = Purple Course Icon (Usually indicates if a course has 2 exits, and only one has been completed in Newer)
- = Shop Course Icon (Represents Shops in Newer)
