NOTE: The BRSTM list was not originally created by me, and was originally created by Skawo (on RVLution). So credit goes to him for the list (and me for adding extra info, and publishing it here).

These are all the BRSTM files in NSMBWii.
They are all located at the /Sound/stream folder on the disc.
They are all controlled by the BRSAR file /Sound/wii_mj2d_sound.brsar on the disc. When replacing any BRSTM file in NSMBWii, you'll need to patch the BRSAR file for the BRSTM to loop properly.
One note to the newbies at NewerSMBW modding; please see Toms' post below (http://horizonwii.net/?page=post&id=738).
