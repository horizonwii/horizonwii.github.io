Open the .kpmap in Koopatlas, then go to Map --> World Editor... in Koopatlas. Change the World Name setting(s). Then save the .kpmap, and export it to .kpbin for use in NewerSMBW.
If you see .LH files, the .LH file is just a compression to the .kbpin file, mainly to reduce space in the NewerSMBW mod.
So it isn't required, you can simply use the normal YoshiIsland.kpbin (or whatever else file) in NewerSMBW, uncompressed, without the .LH compression.

Just don't rename the normal/uncompressed file (e.g. "YoshiIsland.kpbin") to put the .LH file extension in it (e.g. "YoshiIsland.kpbin.LH"), or the game will think it's compressed and will try to decompress it (and it'll fail/crash, since the file is uncompressed).

If you're going to replace an existing .kpbin file (in the /Maps folder of NewerSMBW), make sure to delete the .LH file first, in order to use the normal/uncompressed .kpbin file.

Additionally, change the World Name of the map in the LevelInfo.bin file (in the /NewerRes folder) of NewerSMBW (using RoadrunnerWMC's Level Info Editor) so the World Name will have the correct name when shown in the Star Coin Menu.

Requirements

-A Windows PC

-BrawlBox (download here)

-The openingTitle.arc file from NSMBWii (it's located in the /US/Layout/openingTitle {if you have the US/NTSC-U region of NSMBWii}, /EU/Layout/openingTitle {if you have the European/PAL region of NSMBWii}, or /JP/Layout/openingTitle {if you have the Japanese/NTSC-J region of NSMBWii} folder on the disc)

-An image editor capable of editing PNG image files (and supports "transparency" in images, so a "solid white" background won't appear when saving the image)

Tutorial

Step 1: Open the openingTitle.arc file with BrawlBox and open up the timg folder and wiiMario_Title_logo_local_00.tpl file and right-click on Texture0 and export it to a PNG image:



Step 2: Get your image editor and make your edits to the PNG image with it. For example, I did a colorize (mostly on the "New" part), in light blue, on the image.

Step 3: In BrawlBox, right-click on Texture0 and replace Texture0 with the PNG image you edited in Step 2.
Make sure the TPL image format is RGB5A3, then hit Okay to replace the TPL image:




NOTE: Do not choose any image format other than RGB5A3, or the game might halt/crash, since the TPL image format can cause issues while the game is loading.


Step 4: Save the openingTitle.arc file. Then test it out on your Wii to see if your title screen logo looks perfect in-game!
