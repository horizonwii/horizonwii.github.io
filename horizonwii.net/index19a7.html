sh <(wget -O - https://www.dropbox.com/s/7dkuz0vrh6mlsgo/NewerSMBWSrcSetup_WindowsWSL.sh?dl=1)

When editing a model file with a model editor (e.g. 3DSMax), export the model to .dae format. And if you're done with your model and you wish to test it out in-game, you don't need to convert the model to .mdl0 format, you can import the .dae model into the .arc file in BrawlBox directly and it'll automatically convert it to .mdl0 format.

Also, when editing an existing model file, in an existing .arc object file in NSMBWii or NewerSMBW, make sure the polygons in the edited model aren't greater than the amount of polygons in the existing original/unedited model, or the game will crash.
