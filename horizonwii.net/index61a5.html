Posted by customlevelfan78
Once they are bins they cannot be edited.
Unless if you want to use a hex editor to modify them. Which in most cases, hex editing the NewerSMBW DLCode/DLRelocs/System .bin files isn't a good idea since they can damage the files.

Posted by customlevelfan78
download an IDE for C++ or use Notepad++ which integrates it.
Just keep in mind that the NewerSMBW source files use the Unix (LF) line feed format, so whichever text editor you're using, make sure it supports the Unix (LF) line feed format. Don't use MS Notepad to edit the files, since it doesn't support the Unix (LF) line feed format.
