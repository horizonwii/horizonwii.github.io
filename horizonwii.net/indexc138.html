 1) Activate an ID (how you want, for this example i will use ID 20)
2) Put a Chainer, that is triggered by ID 20 and triggers any ID you want (in this example i will use 21). Make sure to set its Usage Mode to Multi-Use. Set the delay as you wish so that the event stays active the time you want it to.
2) Put a Multi-Chainer, and set the Start Event ID to 21. Now set the End Event ID to 2 numbers next to the Start ID (in this case 23). Set Usage Mode to "Only when Start Event is Active", or the loop will work only once. Also set the delay as you wish.
4) Put a Chainer, and set the Triggering Event ID to 22. Put the Switch Mode to On, then Off after Delay, and the Target ID to 20. This will deactivate Event ID 20. Don't forget to set it to Multi-Use !
5) Put another Chainer, and set this time the Triggering Event ID to 23. Set the Target ID to 20. Don't forget to set this one to Multi-Use, too.

Basically you trigger an event, which triggers another, which triggers 2 events in a sequence : the first deactivates the original event, while the other one reactivates it.

You may ask why is Step 2 needed. Well, if you skip it, you will stop the Multi Chainer while it's running, and the loop will fail.

I hope i explained it well, it's pretty hard.
