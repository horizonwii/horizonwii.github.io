 RiiFS is a simple custom network protocol used by Riivolution to allow a computer to act like a disk filesystem. This allows for both public streamable releases as well as private testing of modifications without needing to restart the game. It can speed up testing a LOT, especially if you simply need to do small fixes.

Requirements :

- A PC (can be Windows, Mac or Linux)
- Wii connected to Wi-Fi
- Good Wi-Fi connection
- RiiFS (see below for download)
- Riivolution v1.03 or higher

Downloading RiiFS

There are two different versions of RiiFS available, the C++ one and the C# one, and both have different features and whatnot.

C++ Version :
Download here

- Works with all the three major OSes
- Has an extra minor update

C# Version :
Download here

- Has an extra "read-only" option
- Windows only
- Requires .NET Framework 2.0 or higher

Step 1 : Setting up the RiiFS folder

First of all, we need to set up the folders for RiiFS to work. Make a folder wherever you want, provided you can type the path later. This folder will work like the SD Card root, so there will be a folder called "riivolution" with XMLs inside, and the rest is up to you.

- Make a folder (for this tutorial, i will use C:\WiiMods as a reference)
- Inside this folder, make a new folder and call it "riivolution" (NO capitals), and paste your XMLs there.
- After this, paste the actual mod files inside "WiiMods"
- Paste RiiFS inside this same folder

Pro Tips :
- You can take a whole NSMBW hack and put it directly in the "WiiMods" folder, but loading times will be LONG, especially for bigger mods like NewerSMBW
- You can even host your mods on a remote server, provided you can run RiiFS on it. You can use it as an easy hub for hack testers, as they receive updates as soon as the files are updated on the server

Step 2 : Getting the IP Addresses

To make RiiFS work, you need both your PC and Wii's IP Addresses. This becomes much easier if both devices are connected to the same network. Make sure you note them !

Wii :
- On the Wii Menu, go in Settings > System Settings (move to page 2) > Internet > Connection Status, the IP will be clearly shown
- Alternatively, open the Homebrew Channel and press the Home button, your IP will be shown near the options

PC (Windows) :
- Type Win+R and type cmd.exe to run the command prompt
- Type ipconfig, the PC's IP will be somewhere inbetween

PC (OS X) :
- Click on the Apple icon on the upper-left corner of the screen.
- Select System Preferences.
- Click Network.
- Your IP address will be listed directly beneath your connection status, in smaller print.

PC (Linux) :
- Open the Terminal and enter the command "ip addr show eth0"

Step 3 : Setting Up the RiiFS XML

As Riivolution will automatically connect to a RiiFS server on a local network, having both your Wii and PC on the same one allows you to skip this step. However, to get Riivolution to connect to a specific or external server it needs an XML file on the SD card to tell it to do so. In the spoiler below is an example and some tips (obviously edit the example to suit your needs) :

<wiidisc version="1"> <id game="SMN" /> <network protocol="riifs" address="192.168.X.X" port="1137" log="false"/> </wiidisc>

XML Explanations :

- You can remove the Game ID restriction to always use RiiFS with every game, provided you have mods for it
- The only protocol available is "riifs", so changing that attribute is useless
- You can edit the port number if you need to, RiiFS defaults to 1137. Make sure to put the same port both on the XML and RiiFS itself !

Pro Tips :

- While you can point to an external server IP, you can also use a domain name (make sure you don't forget the "http://" prefix and point to the correct path !)
- You can add an addional debugging option after the port (log="true"), that will redirect stdout/fwrite and OSReport() over the network, showing these informations in the Terminal.

Step 4 : Starting RiiFS

Now it's time to run RiiFS. Simply double click on the application and it will start running.

It will say "RiiFS Server is now ready for connections on 0.0.0.0:1137" (do not worry about the IP being 0.0.0.0, it will work). If at some point the program says "Return -1", it means that the Riivolution version you have is outdated and needs an update.

You can turn on the Wii with the SD Card inserted and fire up Riivolution, which will now download the files from your PC.

Pro Tips :

- RiiFS can also be run from the Command Prompt/Terminal, with this command (all the options are optional) :
riifs.exe -r [PATH] [PORT]

- Path : You can set the path to the "fake" SD Card root (in this case C:\WiiMods)
- Port : You can set anyone you want, provided it isn't blocked by the router or firewall, default is 1137
- -r (C# Version ONLY) : Enables read-only mode, which doesn't allow files to be changed while RiiFS is running
- To automatize this process, you can easily write a .bat/.sh script using the command inside the spoiler above.

Step 5 : Usage Notes

Some more tips :

- Make sure to not replace any file the game is currently using (e.g do not replace a level while you're playing it), or the game may freeze. To replace a level, first go back to the map and then do it
- This also applies to the rest of the game's content, so use your mind

That's it ! I hope you enjoyed
