I guess you should look into the rel files, that's game code.
Or you can reverse engineer or edit the main.dol file of the NSMBW disc, which is the main code of the game. If you're on Riivolution, you can also use memory offset patch lines in the XML to edit the code (although all the code changes will be made/done in the Wii's RAM/memory).

Keep in mind that you'll need to rip your NSMBW disc with CleanRip on your Wii (http://horizonwii.net/?page=thread&id=56) and open the NSMBW ISO and extract the main.dol file with WiiScrubber v1.40 on your PC to get the main.dol file.

You can use NewerSMBW tilesets, backgrounds, and virtually any assets of NewerSMBW in your own levels or mod. Just don't copy NewerSMBW's levels:
Posted by the Newer Team's NewerSMBW FAQ Page (http://newerteam.com/wii/help/faq.html)
3. Can I use any assets used in the game in my hack?

Anything you'd like aside from the levels. Please give credit, though!

And you'll need to use Reggie! Next or NeweReggie to edit/create NewerSMBW levels, since the original Reggie r3 doesn't necessarily support much of NewerSMBW levels, features, etc., which may be the reason why you're having trouble loading the backgrounds/tilesets in Reggie!

Find Reggie! Next and NeweReggie here: http://horizonwii.net/?page=thread&id=46
