You didn't install it properly, did you followed these steps here?:
Posted by customlevelfan78
Installing PyQt
- Extract the .zip file
- Go to the Python installation folder, then click "Lib" and then "site-packages"
- Create a new folder named "PyQt5", and paste the files there
- Test PyQt5 by opening the terminal and typing "import PyQt5" in it. If nothing happens, everything is installed correctly.

If you did and it still doesn't work, and you're running the 64-bit version of Python/PyQt/etc., try uninstalling Python/PyQt/etc., and try installing the 32-bit (x86/x32/Win32) version of Python/PyQt/etc., and try importing PyQt5 again.
