 How to use the quickpaint tool:
1. Open Reggie to a level.
2. If quick paint options is not already shown, you can enable it in View>Quick Paint Properties. The default keybind is alt-q to enable or disable it.
3. A new window should pop up that says quick paint properties. To start, make sure you have the name of the tileset you want to paint with. For example, if you are making a grassland level, the tileset may be Pa1_nohara.
4. Select a block in your tileset and fill the areas that need to be filled by left clicking to select the tile and then left clicking in the box. For Pa1_nohara, the areas should be filled like so to prevent tiling errors.

5. Click the Add button and name the preset to whatever you want. In previous versions of reggie, you need to name it to the exact name of the tileset (including the pa number). If the tool doesn't work, try to do this first to troubleshoot.
6. Now, you can click the paint button or the erase button and use the right click to drag rainbow blocks all around. After a few seconds, these rainbow blocks will turn into the proper tiles. You can also click the currently selected button again to deselect it instead of having to completely disable quickpaint and re-enable it any time it needs to be used.

This is what the quickpaint tool looks like in the sidebar.
