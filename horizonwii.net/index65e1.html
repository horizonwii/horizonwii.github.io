The .LH file is just a compression to the .arc file, mainly to reduce space in the NewerSMBW mod.
So it isn't required, you can simply use the normal Mario.arc (or whatever else file) in NewerSMBW, uncompressed, without the .LH compression.

Just make sure the filename is "Mario.arc", and don't rename it to put the .LH file extension in it ("Mario.arc.LH"), or the game will think it's compressed and will try to decompress it (and it'll fail/crash, since the file is uncompressed)

You need to make sure that the .dae model file you're importing in BrawlBox is in the Autodesk Collada 2010 format, because BrawlBox is only compatible with importing .dae models in the Autodesk Collada 2010 format.

Get "Autodesk FBX Converter 2010.2" (google it), to convert the "format" of the .dae model file (that you exported with Blender) to the Autodesk Collada 2010 format.
