 Its been a while since I gave an update, eh?


First off, welcome our new staff member John10v10. We hope you can make the Horizon experience one step better at a time.

There have been some recent updates as well. For one, we now have a sprite database! Go to the top bar and click on "Sprite Database" to help document NSMBW.

The forum also has a new look. I've always been inspired by NSMBWHD's look, so I put my own spin on it. I do plan on changing it a bit more, though.

You can now comment on all future Level Depo submissions as well! Whenever you make a new level depo submission, a new thread is made in "Level Showcase" with your submission. This way, you have your very own thread to talk about that level submission. (And yes, you do get a new thread for every submission you make).

The help thread now has new features as well. Visit it in the General NSMBW Hacking forum to check it out!


Logging in glitch
There's a login glitch present at the moment. If you cannot log in and you're absolutely certain that you typed in your username/password correctly, copy the provided error message you get from logging in (yes this is a new feature) and DM one of the staff members from our discord.

Some other glitches have been fixed such as recent posts not working. More bug fixes are expected to come soon.


Discord V Board issue
I've been pushing many updates to help combat the problem between the board and discord.
For one, as you can see, the help thread has gotten updates. You can actually have a bot tell the people on discord that you need help.

Forum report is now a thing as well. When making a thread/post, for example, a bot relays that onto the discord. You can also see that there is another field for making a thread called "Discord Report Message". With this, you can have custom relay messages such as "This is an important update" or "Biggest NSMBW mod in ages, come check it out". It's a small thing, but hopefully you all like it.

I've also wanted to bring up something, which is how you guys should be using the board and discord.

1) Questions should be asked in the help thread, so if anyone else has a similar question they can check there. Hopefully with the new helping features, it can make for a fast and more documented experience.

2) Updates to mods should always be posted in your thread. This way, thorough responses can be mustered up, and more can be shown off. On discord, posting updates isn't the best way of going along with things. Many people may miss the update entirely.

3) Site bugs should be posted in the minor bug thread. I cannot begin to tell you the number of times I've missed bugs presented to me on discord, so to ease things a bit more, please post them there.

4) Ideas and Bugs for Reggie Next Milestone 3 should be in the designated forum. It's easy to document and keep track of passed/rejected ideas and to keep track of bugs.

5) If you want to start a long going conversation, discord isn't your friend for that. Use the board's offtopic forum if you want to create some long-lasting discussions such as "Thoughts on future Nintendo switch launches", "What deserves game of the year" and etc. This also benefits the board, as it brings a much more united community.

Of course, these aren't rules, but instead guidelines. Please follow them to create a much more functional environment.


Along with this I also have future updates I want to put in. These include:

1) A new look to the level depo.
2) More functions to user mods and projects. These include live Q/A sessions, pinned messages, giving project owners the ability to control their own thread, and more.
3) A new tileset and model depo.
4) Cosmetics such as having the site fall when you wish, bats that fly around threads you've posted in, and more. Purchasing these will go straight to funding the board.


Also, an update on Horizon SMBW:
Horizon SMBW's reveal will be postponed from new years/Christmas to a couple months ahead, due to certain events going on at that time. I'm sorry if you were looking forward to a Christmas reveal.


I've also been teetering on making a Youtube channel. If I were to do one, it would be for gaming and mod showcases. We'll see.


Furthermore, if any of you guys know people that are interested in NSMBW hacking, invite them here.


Anyway, that looks like the end of it. Huh, that was a lot of typing.
Wish you guys a nice December and an early Merry Christmas from the Horizon staff
