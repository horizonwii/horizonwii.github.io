 *Tools other than Reggie and Reggie Next.

*None of these tools are mine and I give credit to their
respective creators. Also, huge thanks to Zementblock for
making his Google Drive.



Tanooki Save Editor
This tool lets you modify your NSMBW save. You can decide what world you
begin in, how many of what items you have, what your score is, what item you have,
and how many lives and coins you have.

You can also make the save file a completed one that has all the star coins collected.
This somewhat works with NewerSMBW, but not completely.

Reggie! Dumper
Ever wanted the original files for NSMBW? We can't give them to
you, but you can get yourself with this! It's a Homebrew Channel app that dumps the
data from NSMBW into your SD Card.

It can either dump your entire NSMBW disc, or the Stage Folder with the Texture folder
itself which is required to run Reggie.

Puzzle
This is the main tool used for creating custom tilesets for NSMBW.
It can change the tileset's collision detection, properties, the tileset slot it's on,
and more.

Jigsaw Puzzle
This is a recently updated version of Puzzle speeding up
the saving time, and keeping the animation files when saving. I would suggest using this
over the original Puzzle.

BRFNT2TPL
This tool converts a BRFNT to TPL and vice versa, allowing you to edit a BRFNT font in NewerSMBW/NSMBWii!

BRFNTify Next
BTW, this tool allows you to view a BRFNT file and/or extract textures from it. RoadrunnerWMC (the owner of BRFNTify Next) planned to implement saving of a BRFNT in BRFNTify Next, but it looks like he lost his motivation on it, since it's been a while since he last worked on the project (it was around August 2014).

Staffroll Editor
You know how in the end of NSMBW, blocks show up that say the names of the
creators of the game? You can edit that with this. You need to edit the text file first.
You then use the encode option to put all your edits together.

Koopatlas
In Newer, the World maps are completely custom made. This tool is used to do just that!
It also comes with the World maps from the original NewerSMBW.

Koopatlas Doodads
Koopatlas uses images to add to the environment. All the images
used in Newer can be found in this rar file!

Koopuzzle
Koopatlas also uses custom tilesets specifically made for
world maps. This is a tool you can use to create said tilesets.

Newer Level Info Editor
In Newer, each world has it's own course name,
world name, course id, and much more. This can edit what the course
is called, what the world is called, how many worlds there are, what
world map file to open, decide whether the world is divided into to 2
or not, decide if a course has a secret exit or not, and much much more.

This is probably the second most important tool to modding Newer.

Newer Credits Editor
In Newer, the credits for the game is a lot different than
NSMBW. This tool allows you to edit how/when/what shows up in the Credits Roll!

Be sure to use the Staffroll editor as well.

Newer Message Editor
In Newer, there are blocks that can either give you hints
or messages. This tool can let you either edit old messages or add new ones.

BrawlBox
This program can open, edit and replace all sorts of things. It can open:
-Arcs
-Tpls
-Brstm
-er
-brres

And many more files types. Almost mandatory for Texture hacks.

CTools
It can be used to edit NSMBW .arc (and others) files, too, and more. It also has a BMG editor that allows longer text to be inserted into the NSMBW .bmg message file (which is inside the Message.arc file), unlike the BMG editor in SZS Modifier.

Reggie! Level Editor
Reggie! Next Level Editor

If there are any tools I missed, please let me know!
I'll be sure to update the main post whenever a new
tool comes out or is discovered.
