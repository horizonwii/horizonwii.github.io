 Hello Isaac, I remember you from Youtube. Did you solve the problem by now?

If not then the error might be, that you've installed Dolphin prior to copying NewerDolphin to your drive.
If that's the case, please check -> Config -> Paths, it should read:

Default ISO: empty
DVD Root: C:\NewerDolphin
Apploader: C:\NewerDolphin\apploader.img
Wii NAND Root: C:\NewerDolphin\User\Wii

Then, make sure that
- you've deleted, backed up, or renamed "01-01.arc.LH" to "01-01.arc.LH.ORIG" in c:\NewerDolphin\Stage\
- you've placed your custom level as 01-01.arc into c:\NewerDolphin\Stage\, replacing the original level if needed
- you've started the game by selecting the "boot.dol" file through the "Open..." command (don't start any ISOs/WADs if they are loaded)
