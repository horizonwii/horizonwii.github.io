 This tutorial teaches you how to create a Map Tileset from scratch, starting from a normal level tileset. Credits for the template go to Tempus.

Requirements :

- The tileset you want to convert for use in maps
- Puzzle
- Photoshop (any version above and including CS3 should work)
- Koopuzzle Tileset Template
- Koopuzzle
- BrawlBox

Pre-steps :
1) Open Photoshop
2) Double click NSMB Worldmap Template.atn
3) Open NSMB Worldmap Template.psd

Making the tileset image :
1) Open the tileset you want to convert with Puzzle
2) Click on File > Export Image and save it
3) Open the exported image in Photoshop
4) Take six 'top terrain' tiles, and copy them to the spaces with the blue stars
5) Take twelve 'inner terrain' tiles, and copy them to the yellow starred area
6) Take six 'bottom terrain' tiles, and copy them to the green starred area
7) Create a nice sizeable pattern that tiles to 24x24 for the ground (Keep it bright and colourful and use a geometrically repeating pattern for best effect)
8) Select that pattern (in a box a multiple of 24x24) and go Edit->Define Pattern...
9) With your new pattern, select the Layer 'Put your Pattern Here', and use Edit->Fill->Pattern Fill
10) Run the included Photoshop Action Script, "Make All" (make sure the Folder 'Terrain Bottoms' is open, or it will eventually error)
11) Add deco tiles as desired to any of the red starred spaces
12) Export to PNG

Creating the actual tileset :
1) Open Koopuzzle
2) Click on File > New Tileset from Image and import the png you just made
3) Now it's time to make the objects, check the spoiler if you don't know how to do that (it works similarly to Puzzle).
Click "Add" to add an object and "Remove" to remove the currently selected object.

You can make the object bigger or smaller vertically or horizontally by clicking the appropriate "+" or "-" button.

You can add tiles to an object easily : click on the tile and then click on where you want it to go.

The only other thing to explain here is the menu that reads "Repeat" by default, right next to the "Add" button. It is used to define what happens when you expand the object in Koopatlas. "Repeat" means just repeat the object, like a brick, the "Slope" options tell Koopatlas to treat it like a slope, the "Stretch" options tell it to make one part bigger while keeping the rest the same (for instance "Stretch Center" is used on climbable grids to make the middle bigger while still having the same border), and the other "Repeat" options repeat only one part of the object. Most of the time just plain "Repeat" will be fine.

When you're done making all your objects, check that you have applied the correct tiles to them. Now you can save.

4) Click on File > Save, and give it the name you want. The program will generate an .arc file, which you can use in Koopatlas.

Making your tileset Newer-compatible :

The tileset you just saved is in .arc format, it can be used in Koopatlas, but it CAN'T be used in Newer. To make it compatible follow these steps :

1) Open the .arc file in BrawlBox
2) Look for a file called "optimisedTexture.bin"
3) Export that file and give it the same name as the .arc file (WITHOUT the file extension)
4) The new .bin file can now be moved to the Maps/Texture folder.

As i stated, DO NOT delete the .arc file, or you won't be able to use your tileset in Koopatlas !

That's it, you're done !
