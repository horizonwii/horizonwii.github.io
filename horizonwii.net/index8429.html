This doesn't involve Reggie!, but you can open up an .arc file in SZS Explorer or Brawlbox and replace it there. The files are:
StaffRollTitle.arc (for the credits)
openingTitle.arc (For the Title Screen)

Just search the graphics for them and there you go.

There are those graphics like the "er" and the little slur that gets added to it. Just make them invisible by replacing them with a blank image file at the same size.

If you're talking about the file location of the openingTitle.arc file in NewerSMBW, I think it's located in the OthersE folder (for the US/NTSC-U region of NewerSMBW), OthersP folder (for the European/PAL region of NewerSMBW), and OthersJ folder (for the Japanese/NTSC-J region of NewerSMBW).

Make sure to change the logo in the openingTitle.arc file in all the 3 folders, so that your logo will display for every region of NewerSMBW.
