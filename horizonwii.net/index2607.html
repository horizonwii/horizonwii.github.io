 With this tutorial you can edit colors of liquids in NSMBW Maps (such as World 1) and Sprites (such as Water, Lava...). Please note that this is what i remember about it, so some details might be wrong. If you find any, i would be happy if you reported them.

Requirements :

- The map or object you want to edit
- BrawlBox
- HTML Color Codes

Steps :

- Open the file in BrawlBox
- Go into 3DModels > MDL0 > Material (if a sprite is made by multiple models, do the following for all of them)
- Find the material used by the liquid and select it
- Look for TevColorBlock in the Material
- Go to the linked website and convert the color you wish to use
- Now edit the color codes (DO NOT touch the alpha value)
- Save the file and you're done !

1. You have to edit the CReg0/CReg1 color RGBA values options in the "Materials" of the obj_magmawave_top model in the NSMBW obj_magmawave.arc object file in BrawlBox:

You'll also have to edit the wmy_objmagmawave01 TEX0 texture in the obj_magmawave.arc file as well.


2. You have to edit the CReg0/CReg1/CReg2/KReg0/KReg1/KReg2/KReg3 RGBA values options in the "Materials" of the obj_magma_top model in the NSMBW obj_magma.arc object file in BrawlBox:


Also, you'll have to edit the CReg0/CReg1/CReg2 RGBA values options in the "Materials" of the obj_magma_under model in the NSMBW obj_magma.arc object file in BrawlBox:

You'll also have to edit the wmy_objmagma01 TEX0 texture in the obj_magma.arc file as well.
You have to edit the CReg0/CReg1 color RGBA values options in the "Materials" of the obj_poisonwater_fog model in the NSMBW obj_poisonwater.arc object file in BrawlBox:


You'll also have to edit the CReg0/CReg1/CReg2/KReg0/KReg1/KReg2/KReg3 RGBA values options in the "Materials" of the obj_poisonwater_top model in the NSMBW obj_poisonwater.arc object file in BrawlBox:


You'll also have to edit the CReg0/CReg1/CReg2 RGBA values options in the "Materials" of the obj_poisonwater_under model in the NSMBW obj_poisonwater.arc object file in BrawlBox:

Lastly, you'll also have to edit the wmy_objpoison01 TEX0 texture in the obj_poisonwater.arc file as well.
