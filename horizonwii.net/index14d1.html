 @FruitSmasher it's not there. Check this thread to find original NSMBU high-res rips.

@Buddermario7 : Since 9211tr already resolved one of your questions, i'll stick to the other one.

1) Put a lava sprite in your level.
2) Make an event (if you tell me the conditions i can help you further) and set its "Usage Mode" to "Only While All Events Are Active" so when the conditions is not met anymore the event is deactivated.
3) Now set a Target Event ID to the event activator and put it as the Triggering Event ID in the Lava sprite.
4) Set the Lava "Mode" to choose a speed for the rising lava (and make sure to read the notes for this option)
5) Set a distance for your Lava to move in "Distance"
6) Set the rest of the options how you prefer.
