 A few of us decided to continue the development of Reggie! Next.
It's currently called Reggie! Next Milestone 3

Members

AboodXD - Lead Coder
Grop - Lead Coder
John10v10 - Coder

-The creators of the original Reggie, Reggie Next, NewerReggie, etc (These specific credits are on the Github page)
-customlevelfan78: He made the list of Sprite that require an image, he gave ideas, he made the original list of things to add,
-Zementblock: He made his updated sprite list and gave ideas.
-9211tr: He gave ideas.
-John10v10: He made the Quick Paint Tool, then Stella ported it over to M3!

Here's the current version of Reggie! Next Milestone 3! There are two versions, the source code, and the released version.
The source code is often always the most up-to-date version. How to Run Reggie! Next from Source.

Source Code on Github:
https://github.com/aboood40091/Reggie-Next-M3

Released versions:
v0.3.2
https://github.com/aboood40091/Reggie-Next-M3/releases/tag/v0.3.2

v0.3.1
https://github.com/aboood40091/Reggie-Next-M3/releases/tag/v0.3.1

Running Milestone 3 from source code will most likely be slower than the built version!

Sprite Data:
The latest spritedata.xml can be found here:
http://kemonoart.org/nsmbw/spritedataREWORK.xml
To download, right click on the page and select "Save As."
To install, rename the xml to spritedata.xml, and then place
it in the reggiedata folder, replacing the original file.
