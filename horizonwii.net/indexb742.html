You need to edit Newer's source code. To do that, set up the Newer sources following this tutorial (don't compile the code yet !).

After that, go in NEWERSOURCESFOLDER > Kamek > src and open fileselect.S in Notepad++.

Go to line 330 and edit the name to what you want, then count the characters you've used (including spaces) and add 1 to it, then edit the "#15" next to the text to update it with the new character count.

Next, save, and compile the sources (still following the tutorial linked above)

You have to edit a file named Message.arc. In normal NSMBW, you can find it in EU/XXXEU/Message (in the PAL version, where XXX is one of the languages), US/XXXUS/Message (in the NTSC-U version) and JP/Message (in the NTSC-J version), while in NewerSMBW it's in NewerSMBW/Message.

When you've retrieved this file, open it using CTools' SZS Explorer (you can find the CTools pack in this thread).

After opening the .arc file, double click on the .bmg file. When the editor opens, do your edits and save the BMG (don't forget to save the .arc file after that!)
