What really? That would be very nice?

Yeah. Download LayoutExporter from here.

The program takes an .arc file, extracts the content and converts it : brlyt to rlyt, brlan to rlan and tpl to tga. Run it from the command line with the following :

LayoutExporter.exe export filename outputfolder

Make sure to put the full paths on both the two last parameters (drag the file and output folder in the command line window).

If you want the SDK tools i mentioned, feel free to PM or DM on Discord.
