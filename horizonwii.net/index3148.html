NOTE: I don't remember the tutorial fully, and I didn't backed up the tutorial on RVLution, before it was defunct (I'm rewriting it), so if there's something I missed, please feel free to point it out. I only posted what I have remembered.

Requirements

-A Windows PC

-A text editor

-Treeki's NSMBWii staffroll editor

-A USB or SD Device to launch NSMBWii

-Riivolution and Wii or Wii U

Tutorial

Step 1: Get your NSMBWii staffroll.bin file from the /US/EngUS/staffroll folder of your NSMBWii Disc (folder may be different depending on your region of the game).

Step 2: Get Treeki's NSMBWii staffroll editor here: https://www.dropbox.com/s/vp119axecjcdccd/nsmbw_staffroll_editor.zip?dl=0, and extract it into one folder.

Step 3: Drag your NSMBWii staffroll.bin file and drag it into the staffroll_decode.exe file. You should see a file called staffroll.txt right after.

Step 4: Modify it with a text editor, edit the text. I don't know what the numbers (such as the 8: text, etc.) are for, if anyone knows, please tell me, so don't edit them in the meantime, but edit the text.

Any text surrounded by the
<bold></bold>
are usually used for sections.

Any text surrounded by the
<coin></coin>
are usually used for blocks containing coins, I think...


Step 5: Once you're done making your changes, drag the staffroll.txt file into the staffroll_encode.exe file and there should be a file called staffroll_edited.bin.
Drag that file into your NSMBWii Riivolution patch folder on your SD or USB device (usually it's /nsmb/US/EngUS/staffroll/ [may be different depending on your region/language]), and rename the file to staffroll.bin, then test it to see if it looks good.

NOTE: In NSMBWii, you can simply grab the 01-42.arc file and rename it to 01-01.arc and put it in the Stage folder (and simply replace the existing 01-01.arc) of your Riivolution patch folder to rapidly test the staffroll.bin out (by simply entering World 1-1)!
