If it's old (like that old [and long-gone] Dell Inspiron 8600 laptop my parents had when I was a little kid), any newer version of Windows may run very slow (or not install at all, or if so, in most cases, it'll run incorrectly, since you may not be able to find the appropriate drivers to get it working perfect and such, since the laptop is extremely old).

So if he's on an old computer, I would instead recommend re-downloading Reggie! and if that doesn't work, instead use Reggie! Next and if that doesn't work, re-download Reggie! Next, or I don't know the answer to the problem.
