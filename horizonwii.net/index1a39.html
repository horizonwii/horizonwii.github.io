You were attempting to extract the ISO file (from your Wii game disc) to your SD card.

The reason you're getting that "writing error", is because Wii ISO files are 4.37 GB in filesize (and your SD card can only hold 2 GB), so there's not enough free space to extract the ISO file to your SD card (and as a result, CleanRip will corrupt your SD card, resulting in the ISO file not getting deleted {by clicking "Delete"} unless if you scan/repair the SD card).

You cannot use your SD card to extract the ISO file. You'll need to use an external USB hard drive with at least 6 GB or higher (plugged into one of the Wii's USB ports), formatted in the NTFS filesystem in order to extract the ISO file (from the Wii game disc) with CleanRip.

The .LH file is just a compression to the .arc file, mainly to reduce space in the NewerSMBW mod.
So it isn't required, you can simply use the normal 01-01.arc (or whatever else file) in NewerSMBW, uncompressed, without the .LH compression.

Just don't rename the normal/uncompressed file (e.g. "01-01.arc") to put the .LH file extension in it (e.g. "01-01.arc.LH"), or the game will think it's compressed and will try to decompress it (and it'll fail/crash, since the file is uncompressed).
If you didn't rename the 01-01.arc file to 01-01.arc.LH, then you should look at the Level Troubleshooting Guide , for help.
